# SW 알고리즘 특강

* 주제 : 코딩 테스트 준비를 위한 알고리즘 특강 with 파이썬
* 대상 : 대학 SW 전공자 또는 SW 분야 취업을 위해 코딩 테스트를 준비하는 초보자

* 과정
  - 코딩 테스트 개요
  - 파이썬 기초 실습
  - 알고리즘 이론
    - 그리디 greedy
    - 정렬
    - 탐색
    - 이진탐색
    - 다이나믹 프로그래밍
    - 그래프
    - 최단경로


* 교육 방법
  - 이론 + 실습 + 그룹 스터디
